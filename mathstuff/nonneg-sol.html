Counting Using One-To-One Correspondence (I): Distributing Peaches

Suppose that we have $n$ identical peaches and $k$ different people. How many ways are there to distribute the peaches? This is the same as asking the number of nonnegative integer solutions $(x_1, x_2, \ldots, x_k)$ to the equation $$x_1 + x_2 + \ldots + x_k = n,$$ where $x_i$ represents the number of peaches the $i$-th person receives. For example, when $n=4$ and $k=3$, the equation becomes

$x_1 + x_2 + x_3 = 4$,

and all the nonnegative integer solutions to it are:

(0,0,4)    (0,1,3)    (0,2,2)    (0,3,1)    (0,4,0)
(1,0,3)    (1,1,2)    (1,2,1)    (1,3,0)    (2,0,2)
 (2,1,1)    (2,2,0)    (3,0,1)    (3,1,0)    (4,0,0).

These solutions represent all 15 ways of distributing 4 identical peaches to 3 people. How do we know that there are only 15 solutions though? Let's count. Say we decide to give 2, 1 and 1 peaches to the first, second and third person, respectively, which corresponds to the solution (2,1,1). We can do the following. First, count two peaches, and put a stick indicating that these go to the first person, then count another peach, and put a stick showing that this peach goes to the second person, and then the remaining peach goes to the third person. Pictorially, if we use ♥ to represent a peach and a vertical bar to represent a stick, then our distribution scheme (2,1,1) is represented by ♥♥|♥|♥. Two other examples showing this correspondence are

(3,1,0) $\rightleftharpoons$ ♥♥♥|♥|    and    (0,2,2) $\rightleftharpoons$ |♥♥|♥♥.

Observe that every distribution scheme is represented uniquely by such a heart-bar diagram, and vice versa. In other words, there is a one-to-one correspondence between the set all distribution schemes and the set of all such heart-bar diagrams, so the number of nonnegative integer solutions to $x_1 + x_2 + x_3 = 4$ (which represent the distribution schemes) can be obtained by counting the heart-bar diagrams with four hearts and two bars! (This is because we have four peaches and we need two bars to separate three people.) Of course, it is just the number of ways to choose two slots out of six to put the bars, which is $${6 \choose 2} = 15,$$ so our 15 solutions above are indeed complete!

When working with the equation $x_1 + x_2 + \ldots + x_k = n$ in general, with the same reasoning, we need $k-1$ bars to separate the $n$ hearts (or peaches) for $k$ unknowns (or people), so the number of nonnegative integer solutions to the equation is equal to the number of heart-bar diagrams with $n$ hearts and $k-1$ bars. In summary,

If $n$ and $k$ are positive integers, then the number of nonnegative integer solutions to the equation $x_1 + x_2 + \ldots + x_k = n$ is $${n+k-1 \choose k-1}.$$
